# maimai-timing-align

基于 Streamlit 的舞萌视频对齐工具：
- 自动检测两段视频中的“明显转场”（不要求固定为亮→暗）
- 按时间戳（兼容不同帧率/VFR）对齐
- 从锚点开始保留 Clip1 画面，混合 Clip1+Clip2 音轨并导出完整视频

## 环境

- Python 3.14
- `uv`
- `ffmpeg` / `ffprobe`（需在 PATH 中）

## 安装

```bash
uv sync
```

## 启动

```bash
uv run streamlit run streamlit_app.py
```

## 使用说明

1. 上传 Clip1（作为输出画面）和 Clip2（用于对齐 + 混音）
2. 调整参数（可先使用默认）
3. 点击“分析转场并导出完整视频”
4. 下载导出结果

## 输出规则

- 画面：仅使用 Clip1
- 音频：Clip1 和 Clip2 按检测偏移对齐后混音
- 时间范围：从共同锚点开始，到两段可重叠尾段结束

## 锚点选择策略

- 转场检测不区分亮→暗或暗→亮，只要是“明显突变”都会计入候选。
- 候选峰会进行加权：
	- 更靠前时间位置权重更高（避免把开头匹配到结尾）
	- 对“前面已有近邻峰”的候选做轻微加权（更偏向前段切出处）
- Clip1 和 Clip2 使用不同权重机制：
	- Clip1（实拍）偏稳健，保留较强前段偏置
	- Clip2（内录）偏向“前段第二峰（切出）”，并对“首峰”追加惩罚
- 你可以在侧边栏「高级：锚点权重」中微调参数。

## 常见问题

- 检测失败：可降低 `min_peak_z` 或增大 `smoothing_window`
- 置信度低：建议开启手动锚点输入进行覆盖
- 导出失败：确认 `ffmpeg -version` 和 `ffprobe -version` 可执行
